 HR Analytics Dashboard

Overview

This project analyzes employee attrition data using Excel and Power BI. It includes visual insights on attrition by department, gender, education, age group, and experience level. The goal is to help HR teams identify key factors influencing employee turnover and improve retention strategies.

 Tools Used
- Microsoft Excel for data cleaning and preprocessing  
- Power BI for data visualization and dashboard creation  
- Kaggle dataset: HR Attrition & Performance data  

Project Steps
1. Cleaned and formatted HR data in Excel.  
2. Imported the dataset into Power BI.  
3. Created DAX measures such as Total Employees, Total Attrition, Attrition Rate, and Average Age.  
4. Built interactive visuals analyzing attrition across various dimensions like department, gender, job role, education field, age group, and experience level.  
5. Designed a dashboard with a clean layout and meaningful insights.

 Key Insights
- Higher attrition among younger and entry-level employees.  
- Sales and HR departments show relatively higher turnover.  
- Overtime is linked to increased attrition rates.  
- Mid-level experienced employees tend to stay longer.
